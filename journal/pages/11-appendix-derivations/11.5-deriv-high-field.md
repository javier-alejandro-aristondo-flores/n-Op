# Non-Equilibrium, High-Field, Hot-Carrier Physics for UWBG Chips

> **Provenance rule.** Coefficient values below carry `[Wave-1 …]` tags where the
> 2026-06-10 III-N audit corrected them (`[timeline] §2026-06-10 (Wave 1)` §3).
> **Seed coefficients from the accuracy-ledger / reference-data CSVs, never from this
> file's tables** — the CSVs are the audited canonical values.

**Scope.** Translate the physics that governs UWBG semiconductors in jet-turbine-class operating envelopes (T_L ≥ 500 °C, E up to several MV/cm, current densities pushing the velocity-saturation regime, possible vibration / radiation overlays) into the typed-formula / typed-residual vocabulary already established for `/physics`. Materials anchor set: **diamond (C), c-BN, h-BN, AlN, GaN, β-Ga₂O₃, AlGaN ternary, plus refractory metals (W, Mo, Re, Ir, TiN, TaN) and standard dielectrics (SiO₂, Al₂O₃, HfO₂)** as contact/dielectric counterparts. Diamond is the non-polar anchor (no Fröhlich term); the other UWBG anchors are polar.

CS-discipline reminder applied throughout: every quantity below is presented as `name : (typed inputs) → typed output`, every residual as `‖LHS − RHS‖² over typed domain`, and every "method" is a sub-method already in the closed 12 (or, where genuinely new, flagged with a recommendation).

## Part A — High-Field Transport Regimes

### A.1 Operational regime table

| Regime | Field window (rough, diamond) | Dominant physics | Governing object |
|---|---|---|---|
| Ohmic | E ≲ 10⁴ V/cm | Linear response; μ from impurity + acoustic + optical phonons | Drift-diffusion, σ = enμ |
| Warm-carrier | 10⁴ – 10⁵ V/cm | T_e begins to rise above T_L; μ slightly sublinear | Energy-balance BTE, RTA still adequate |
| Hot-carrier / pre-saturation | 10⁵ – 10⁶ V/cm | T_e ≫ T_L; v drifts toward v_sat; μ(E) collapses | Energy + momentum BTE, Caughey-Thomas envelope |
| Velocity-saturated | E ≳ a few × 10⁵ V/cm | v ≈ v_sat; j ≈ q n v_sat (independent of E) | v_sat parameterization |
| Avalanche (pre-breakdown) | approaching E_BR | Impact ionization rate non-negligible; carrier multiplication | Chynoweth α(E) |
| Breakdown | E ≥ E_BR | M → ∞; junction fails | Avalanche-integral condition |

Approximate field anchors per material (300 K; weakly T-dependent):

| Material | μ_e (cm²/Vs) | v_sat (cm/s) | E_BR (MV/cm) | Notes |
|---|---|---|---|---|
| Diamond | ~2000 (intrinsic) | ~1.5×10⁷ | 10–20 | Non-polar; acoustic + intervalley dominate |
| c-BN | ~825 | ~1.5×10⁷ | ~17.5 | Polar but weak; large gap |
| h-BN | anisotropic | – | ~12 | Layered, anisotropic transport |
| AlN | 871(⊥)/619(∥) intrinsic FP; best exp 426 [Wave-1 A13] | ~1.4×10⁷ [Wave-1] | ~12 | Strong polar |
| GaN | ~1500 | 1.4×10⁷ (v_peak 2.85×10⁷) [Wave-1 A11] | ~3.5 | Strong polar |
| β-Ga₂O₃ | ~200 | ~2×10⁷ | ~8 | Strong polar, low μ, low κ |
| Al_xGa_{1-x}N | interp. | interp. | 3.5→12 | Linear-ish interpolation, bowing for gap |

### A.2 Field-dependent mobility — Caughey-Thomas

**Named formula (add to registry):** `field-dependent-mobility`

```
field-dependent-mobility :
  ( μ₀ : ScalarPositive,            ; low-field mobility, T- and doping-dependent
    E  : ScalarPositive,            ; local |E|
    v_sat : ScalarPositive,         ; saturation velocity
    β  : ScalarPositiveSmallInt )   ; material exponent (1 for electrons in GaAs-like, ~2 for diamond/GaN)
  → ScalarPositive                  ; μ(E)
```

Definition (literal): μ(E) = μ₀ · [1 + (μ₀ E / v_sat)^β]^(−1/β).

- **Cheap path.** Direct evaluation; β table per material; μ₀(T, N_D) via a second named formula `mobility-impurity-phonon` (Caughey-Thomas-Klaassen style with T^(−n) phonon term + Brooks-Herring ionized-impurity term).
- **Faithful path.** μ₀ extracted from the converged BTE `kinetic-evolution-of` solve (sub-method `BTE-RTA` already on the menu); the Caughey-Thomas form is then verified against the BTE-derived j(E) curve as an **algebraic-identities** residual (a method-equivalence: ChevyT(E) ≡ BTE-j(E)/(n q E)).

### A.3 Saturation velocity v_sat

Set by polar-optical-phonon (POP) emission rate (polar materials) or by intervalley scattering / acoustic-deformation-potential (diamond). Heuristic ceiling: v_sat ≈ √(2 ℏ ω_LO / m*) for POP-limited transport (gives ~2×10⁷ cm/s for GaN with ω_LO ≈ 92 meV, m* ≈ 0.2 m_e — within factor of order unity of measured).

**Named formula:** `v-sat-POP-limit` `(ℏω_LO, m*) → ScalarPositive` for polar UWBG.
**Named formula:** `v-sat-intervalley` `(ΔE_valley, m*, τ_iv) → ScalarPositive` for diamond.

Cheap path: closed form. Faithful path: extract v_sat as the high-field asymptote of the BTE-driven j(E) curve; residual = (v_sat^cheap − v_sat^BTE)².

### A.4 Worked example (diamond, 800 K, E = 5×10⁵ V/cm)

μ₀(800 K) ≈ 2000 × (300/800)^(2.5) ≈ 175 cm²/Vs (phonon-limited scaling).
Caughey-Thomas with v_sat = 1.5×10⁷, β = 2:
μ₀E/v_sat = 175 · 5×10⁵ / 1.5×10⁷ ≈ 5.83.
μ(E) ≈ 175 / [1 + 5.83²]^0.5 ≈ 29.5 cm²/Vs.
Drift velocity v_d = μE ≈ 1.48×10⁷ cm/s — already saturated. This is the regime jet-turbine UWBG devices live in.

## Part B — Polar-Optical-Phonon (Fröhlich) Scattering

Critical for AlN, GaN, β-Ga₂O₃, c-BN, AlGaN. **Not** for diamond (non-polar; acoustic-deformation-potential + intervalley dominate; h-BN is borderline-polar in plane).

### B.1 Fröhlich coupling

**Named formula:** `polar-coupling-constant`

```
polar-coupling-constant :
  ( ε₀ : ScalarPositive,         ; static permittivity
    ε∞ : ScalarPositive,         ; high-frequency permittivity
    ω_LO : ScalarPositive,       ; LO phonon angular frequency
    m_eff : ScalarPositive,      ; effective mass
    q_e : Constant )
  → ScalarPositive               ; α (dimensionless)
```

Definition: α = (q²/4πℏ)·√(m*/(2ℏω_LO))·(1/ε∞ − 1/ε₀).

Anchors (electrons):

| Material | ω_LO (meV) | ε₀ | ε∞ | m*/m_e | α |
|---|---|---|---|---|---|
| GaN | 92 | 8.9 | 5.35 | 0.20 | ~0.40 [Wave-1 A12] |
| AlN | 110–114 [Wave-1 A9] | 8.5 | 4.77 | 0.32 | ~0.58 [Wave-1 A12] |
| β-Ga₂O₃ | 44 (avg) | 10.2 | 3.6 | 0.28 | ~1.7 |
| c-BN | 161 | 6.8 | 4.5 | 0.35 | ~0.34 |
| Diamond | 165 (TO) | 5.7 | 5.7 | 0.36 | 0 (non-polar) |

### B.2 POP scattering rate

**Named formula:** `polar-optical-scattering-rate` (per state, energy ε):

```
polar-optical-scattering-rate :
  ( α : ScalarPositive,
    ω_LO : ScalarPositive,
    ε : ScalarPositive,          ; carrier energy
    T : ScalarPositive,
    branch : { 'emission, 'absorption } )
  → ScalarPositive               ; 1/τ_PO
```

Standard expression with Bose-Einstein occupation N_LO = [exp(ℏω_LO/k_BT) − 1]^(−1); emission threshold at ε ≥ ℏω_LO.

- **Cheap path.** Closed-form 1/τ_PO(ε,T) with the standard Fröhlich formula; mobility via μ_PO = q⟨τ_PO⟩/m*.
- **Faithful path.** Same rate enters the BTE collision integral; converged BTE-RTA mobility is the gold value; residual = (μ_PO^cheap − μ_PO^BTE)².

### B.3 T-dependence of PO-limited mobility

For ℏω_LO ≫ k_BT (true at 300 K for GaN/AlN but **not** at 800 K for β-Ga₂O₃ where ℏω_LO ≈ 44 meV ≈ k_B·500 K), μ_PO ∝ [exp(ℏω_LO/k_BT) − 1]. This is **the** load-bearing reason β-Ga₂O₃ mobility collapses much harder than GaN with T — it sees the high-T side of its Fröhlich curve in turbine conditions.

## Part C — Breakdown Physics

### C.1 Chynoweth's law

**Named formula:** `impact-ionization-coefficient`

```
impact-ionization-coefficient :
  ( E : ScalarPositive,
    a : ScalarPositive,            ; material/orientation constant
    b : ScalarPositive,            ; activation field
    carrier : { 'electron, 'hole } )
  → ScalarPositive                 ; α (cm⁻¹)
```

α(E) = a · exp(−b/E).

Material anchors:

| Material | a_n (cm⁻¹) | b_n (V/cm) | a_p (cm⁻¹) | b_p (V/cm) |
|---|---|---|---|---|
| Diamond | 1.93×10⁵ | 7.59×10⁶ | 1.93×10⁵ (e/h never separated) | 7.59×10⁶ |
| GaN | 4.48×10⁸ [Wave-1 A14] | 3.39×10⁷ | 7.13×10⁶ | 1.46×10⁷ |
| AlN | 8.875×10⁶ (Bulutay MC, electron-only) | 3.759×10⁸ | measured = GAP (cert-refused) | — |
| β-Ga₂O₃ | anisotropic — quoted as E_c 10.2/4.8/7.6 MV/cm (Ghosh–Singisetti) | — | holes never measured | — |
| 4H-SiC (ref) | 1.88×10⁶ | 9.13×10⁶ | — (per-carrier split lands with Wave 3) | — |

(Mirrors the audited canonical set — the `impact-ionization-*` rows of
`physics/library/cert/reference-data/transport-coefficients.csv` and the accuracy-ledger
Chynoweth table. AlN measured α_ii and the 4H-SiC per-carrier split are GAP.)

### C.2 Avalanche multiplication and breakdown condition

**Named formula:** `avalanche-multiplication`

```
avalanche-multiplication :
  ( α_n : Field[x ∈ [0, W]],
    α_p : Field[x ∈ [0, W]],
    W : ScalarPositive )
  → ScalarPositive                 ; M
```

For equal-α: M = 1 / (1 − ∫₀ᵂ α(E(x)) dx). The **breakdown integral condition** is ∫α dx = 1.

This integral over a *spatial field* implies the breakdown observable lives naturally in the **field-on-grid interface** integrated to a **scalar** — same shape as a 1D Poisson solve.

### C.3 Temperature dependence of E_BR

In wide-gap materials E_BR has a **positive** dT/dT_L (unlike Si where it can be near-flat): hot lattice → more phonons → shorter mean free path between ionization events → α at fixed E decreases → E_BR rises. Often parameterized as E_BR(T) = E_BR(300) · (1 + κ_BR (T − 300)) with κ_BR ≈ +5×10⁻⁴ /K for diamond, ~+7×10⁻⁴ /K for 4H-SiC. **Important for jet-turbine design**: device hardens against breakdown as it heats, but mobility collapses, so the constraint trades.

### C.4 Zener tunneling

For UWBG (E_g ≥ 3.5 eV) Zener tunneling becomes meaningful only at extreme fields (≳10⁷ V/cm) in very thin junctions or under high reverse bias on heterojunctions with strong polarization fields. Catalog the formula but flag it second-priority:

**Named formula:** `interband-tunneling-rate` `(E_g, m_r*, E) → ScalarPositive` with the standard Kane prefactor and exp(−π m*^(1/2) E_g^(3/2) / (2√2 q ℏ E)) suppression.

### C.5 Cheap vs faithful

- Cheap: Chynoweth α(E) closed-form; avalanche integral via Gaussian quadrature over a 1D E(x).
- Faithful: ionization coefficient extracted from BTE high-energy tail of the distribution function (sub-method `BTE-full` already on the closed menu); residual: cheap Chynoweth fit against BTE-derived α at sample fields. **Algebraic-identities residual: method-equivalence** between Chynoweth and BTE in the high-field asymptotic regime.

## Part D — Self-Heating and Coupled Electrical-Thermal

This is the central failure-mode coupling for jet-turbine UWBG: the device that survives breakdown still has to dissipate the heat it generates without thermal runaway.

### D.1 Heat-flow equation

PDE (cell-resolved):

C_p ρ ∂T/∂t = ∇·(κ(T) ∇T) + Q,    Q = **j**·**E**.

In the GENERIC view this is precisely the M-driven evolution of the **lattice temperature aspect of the unified state**. Specifically: the Joule term j·E is the dissipated power that drives ∂T/∂t through the entropy production δS/δx route; the diffusion term is the symmetric-PSD spatial coupling.

### D.2 Thermal conductivity κ(T)

**Named formula:** proposed here as `phonon-kappa-T`; it landed split across `single-mode-rta-lattice-kappa` (row 25) for κ_L itself and `kappa-4phonon-high-t-correction` (row 121) for the high-T falloff, rather than as one fitted power law. The fit below is the survey-grade form:

κ(T) = κ_0 · (T_0/T)^n,     n ≈ 1.25 for diamond (300 → 600 K), n ≈ 1.0 for GaN.

| Material | κ(300) [W/m·K] | κ(600) | κ(900) | κ(1200) |
|---|---|---|---|---|
| Diamond | 2200 | ~700 | ~400 | ~280 |
| c-BN | 1300 | ~500 | ~300 | ~210 |
| AlN | 285 | ~140 | ~95 | ~75 |
| GaN | 130 | ~75 | ~55 | ~45 |
| β-Ga₂O₃ (⊥b) | 11 | ~7 | ~5.5 | ~4.5 |
| h-BN (in-plane) | 400 | ~190 | ~130 | ~100 |

Cheap path: closed-form Slack fit. Faithful path: BTE-RTA on phonon distribution (already in 12-method menu; `kinetic-evolution-of` with `BTE-RTA` over phonons; `linear-response` for Green-Kubo). Residual: method-equivalence between Slack-fit κ(T) and the BTE-derived κ at training points.

### D.3 Thermal boundary resistance (TBR)

Critical at refractory-metal / UWBG and dielectric / UWBG interfaces. Two cheap models:

**Named formula:** `acoustic-mismatch-TBR`
**Named formula:** `diffuse-mismatch-TBR`

Inputs: `(ρ₁, c₁, ρ₂, c₂, T)` for AMM; phonon DOS for DMM. Both return a TBR (m²·K/W) used as an interface Robin boundary condition on the heat PDE.

Anchors (diamond / W ≈ 1×10⁻⁸ m²K/W; GaN / SiC ≈ 3×10⁻⁹ m²K/W; β-Ga₂O₃ / Cu ≈ 2×10⁻⁸ m²K/W).

### D.4 Coupled electrical-thermal residual (the key new residual category)

The standalone EOM-violation residual already covers ‖dx/dt − (L δE/δx + M δS/δx)‖². But the operationally critical question for chip design is whether a **predicted joint (E(r), j(r), T(r)) field** simultaneously satisfies three PDEs:

1. Poisson: ∇·(ε ∇φ) = −ρ
2. Drift-diffusion (or BTE): ∂n/∂t + ∇·j = G − R
3. Heat: C_p ρ ∂T/∂t − ∇·(κ ∇T) = j·E

**Recommendation: add a new residual sub-category `coupled-pde/`** under the existing `residuals/` tree, distinct from `eom-violation/`. It exists because:

- `eom-violation` is over the *unified-state evolution* and is GENERIC-structural.
- `coupled-pde` is over the *emergent field tuple* (φ, n, j, T) and is what the PINO loss actually penalizes for chip-design observables.

The cleaner alternative is to keep it inside `eom-violation/` with a dedicated `coupled-em-thermal-pde-residual` *sub-residual* — preserves the principle "one primary residual" but parameterizes by the field tuple under test. Use this if Stream-J cross-stream synthesis prefers single-category.

```
residuals/eom-violation/coupled-em-thermal-pde-residual :
  ( φ : Field[ℝ³],
    n : Field[ℝ³],
    j : Field[ℝ³ → ℝ³],
    T : Field[ℝ³],
    material : MaterialRecord,
    environment : Environment )
  → ScalarNonNegative
```

## Part E — Hot-Electron Temperature Equation

UWBG materials have *large* ℏω_LO, so τ_E (energy-relaxation time) is *long* compared to standard III-V or Si — meaning T_e can substantially overshoot T_L under modest fields. Quantitatively: τ_E ~ ℏω_LO / (2 k_B T_L · sinh(ℏω_LO/2k_BT_L)) × (typical 10⁻¹³ s).

### E.1 Two-temperature energy balance

**Named formula:** `hot-carrier-temperature-balance`

```
hot-carrier-temperature-balance :
  ( n : ScalarPositive,
    j : Vector[3],
    E_field : Vector[3],
    T_L : ScalarPositive,
    T_e : ScalarPositive,
    τ_E : ScalarPositive )
  → ScalarSigned                   ; LHS - RHS of energy balance
```

Steady-state closed form: T_e − T_L = (2/3)·(j·E)·τ_E / (n k_B).

### E.2 τ_E parameterization

**Named formula:** `tau-energy-POP-acoustic` (row 73). Proposed here as two formulas —
`tau-energy-pop` `(ℏω_LO, T_L) → ScalarPositive` for polar UWBG and `tau-energy-acoustic`
`(v_s, m*, T_L) → ScalarPositive` for diamond — they landed as one row carrying both
channels, `(α_F, v_s, ρ, Ξ, T) → τ_E`.

| Material | τ_E (300 K) | τ_E (800 K) |
|---|---|---|
| GaN | ~0.2 ps | ~0.05 ps |
| AlN | ~0.15 ps | ~0.04 ps |
| Diamond | ~0.5 ps (acoustic-dom) | ~0.15 ps |
| β-Ga₂O₃ | ~0.1 ps | ~0.03 ps |

### E.3 Worked example (GaN HEMT channel)

n = 1×10¹³ cm⁻², j = 2 A/mm (channel), E ≈ 4×10⁵ V/cm, T_L = 800 K, τ_E ≈ 0.05 ps.
2D: Joule density (j_2D · E)/t_ch with t_ch ≈ 5 nm → ~1.6×10²¹ W/m³.
T_e − T_L ≈ (2/3)·1.6×10²¹ · 5×10⁻¹⁴ / (10¹⁹ · 1.38×10⁻²³) ≈ 386 K.
**T_e ≈ 1186 K with T_L ≈ 800 K** — physically: avalanche threshold reached with sub-breakdown E because the *carriers* are hot enough to ionize. This is why hot-carrier modeling matters more for UWBG-at-T than for cold UWBG.

### E.4 State-vector status

T_e is **emergent** (it is the second moment of the BTE distribution about the band edge — already derivable from the converged f_n(k)). Keep it out of the irreducible state vector; expose as a temperature-resolved observable. T_L, in contrast, *is* present already (encoded in vibrational entropy / phonon distribution via S[x] in the unified functional).

## Part F — Field-Emission and Tunneling at Contacts

### F.1 Fowler-Nordheim

**Named formula:** `field-emission-current`

```
field-emission-current :
  ( E : ScalarPositive,
    φ_B : ScalarPositive,         ; Schottky barrier
    m_eff : ScalarPositive )
  → ScalarPositive                 ; J (A/m²)
```

J_FN = (q³ E² / 8π h φ_B) · exp(−4√(2m*) φ_B^(3/2) / (3ℏ q E)).

### F.2 Thermionic + thermionic-field emission

**Named formula:** `thermionic-emission-current`
**Named formula:** `thermionic-field-emission-current`

The Padovani-Stratton TFE crossover parameter E_00 = (ℏ/2)·√(N/(m*·ε)) distinguishes regimes: E_00 ≪ k_BT (TE), E_00 ≈ k_BT (TFE), E_00 ≫ k_BT (FE).

For diamond Schottky contacts at 800 K with N_D = 10¹⁸: E_00 ≈ 8 meV, k_BT ≈ 69 meV → **pure thermionic** dominates at T. At 300 K with high-doping pulse (10²⁰) → TFE regime.

### F.3 Image-force barrier lowering

**Named formula:** `image-force-barrier-lowering`

Δφ = √(qE/(4πε_s)).

For diamond ε_s ≈ 5.7 ε₀ at E = 10⁶ V/cm: Δφ ≈ 0.16 eV [Wave-1 A15 reconciliation; 0.50 eV at 10⁷ V/cm] — non-trivial relative to typical Schottky barriers (~1–2 eV on diamond).

### F.4 Method assignment

All four belong to **`response-of-to`** template family: the observable is current responding to applied bias under boundary conditions. Sub-method: a new `interface-tunneling` evaluator built on existing `linear-response` plus an explicit barrier model. Faithful path: NEGF tunneling probability (out-of-scope here, route to Stream that owns DFT/NEGF substrate).

## Part G — Hydrodynamic Phonon Transport

Relevant niche for high-purity diamond (and possibly h-BN) at intermediate T where N (Normal, momentum-conserving) processes dominate over U (Umklapp, momentum-relaxing). Operationally requires *splitting* the phonon collision operator:

C[f] = C_N[f] + C_U[f].

Two named regimes:

- **Drift state** (Poiseuille phonon flow): τ_N ≪ L/v_s ≪ τ_U → momentum-conserving drift; κ enhanced superlinearly with sample width.
- **Second sound**: temperature waves with finite speed c_2 = v_s/√3.

**Named formula:** `phonon-poiseuille-length` `(τ_N, τ_U, v_s, L) → ScalarPositive` predicting onset.
**Named formula:** `second-sound-speed` `(v_s, dim) → ScalarPositive`.

These are diamond-special edge cases (observed at ~70 K in high-purity diamond; at 800 K U dominates so this regime collapses). Catalog for completeness; mark as **low-priority for jet-turbine envelope** but high-priority if the project extends to cryogenic harsh environments (space, deep-cooled detectors).

Methods assignment: extension of `kinetic-evolution-of` `BTE-full` with a structured collision kernel — no new method primitive needed, just a typed `CollisionOperator` parameter that admits the (C_N, C_U) split.

## Part I — Vibration / Mechanical Stress Effects on Transport

Jet-turbine specific. Cyclic mechanical stress modulates mobility, gap, and defect generation.

### I.1 Piezoresistance

**Named formula:** `piezoresistance-coeff`

Δρ/ρ = π_ijkl σ_kl.    π_ijkl is the piezoresistance tensor (independent measurements per material; diamond π_11 ≈ +4×10⁻¹¹ Pa⁻¹).

### I.2 Stress-induced bandgap shift

**Named formula:** `deformation-potential-gap-shift`

ΔE_g = Ξ_c · ε_hydro + Ξ_v · ε_hydro + (uniaxial and shear terms).

Diamond Ξ ≈ −7 eV (negative deformation potential — compressive strain *lowers* gap; rare). GaN Ξ_c ≈ −9 eV, Ξ_v ≈ +2 eV.

### I.3 Cyclic-loading defect generation

Beyond the elasticity limit, Coffin-Manson-style fatigue produces new vacancy/interstitial pairs. Couple to defect-formation-energy already in the registry; new **named formula:** `plastic-strain-fatigue-life` `(Δε_p, ε_f, c) → CyclesToFailure`.

### I.4 Method assignment

All three sit in **`response-of-to`** with the applied stress as `ExternalField`; the existing `applied_stress` slot in `Environment` already exposes the input cleanly. The output is a **tensor-indexed** observable (modulated mobility / conductivity tensor under stress) — bundle-fit.

### I.5 Recommendation

Add a single named formula for the *combined* effect: `transport-under-stress` `(σ_ext, π_tensor, μ_0, T) → MobilityTensor` — the cheap path. Faithful: BTE-RTA evaluated against a strain-deformed band structure (composition of existing primitives; no new method primitive needed).

### J.2 Observable bundles — additions

The existing observable bundles already cover most outputs:

- κ(T), μ(T,E), v_sat(T) → **temperature-resolved** + **tensor-indexed**
- Mobility tensor under stress → **tensor-indexed**
- α(E), J_FN(E), Schottky J-V → no clean bundle exists today

**Recommendation: add two new bundles.**

```
observables/
├── field-resolved/             α(E), J_FN(E), v(E), M(E)
│                               outputs are fields over the E-field axis
│
└── hot-carrier-resolved/        T_e(E,T_L), τ_E(T_L),
                                f(k) departure from f⁰
                                outputs parameterized by carrier energy / k
```

These mirror `energy-resolved/` and `bz-resolved/` exactly in shape; the BZ-integration and FFT machinery already there is reusable. `field-resolved` implements the **field-on-grid** + **response** interfaces (causality not required — these are real-valued nonlinear responses, so a relaxed `response` variant should be added or the existing interface needs a `causal? : Boolean` parameter).

### J.3 Method assignment

All Part-A through Part-I physics maps cleanly into the **existing 12 methods**. No new primitives required:

| Stream-S4 physics | Method | Sub-method |
|---|---|---|
| Caughey-Thomas, Chynoweth, Fröhlich rate, FN, image lowering, NRT | `algebraic-combination` / `algebraic-of` | direct named-formula eval |
| v_sat from BTE high-field limit | `kinetic-evolution-of` | `BTE-RTA` |
| α(E) from BTE distribution tail | `kinetic-evolution-of` | `BTE-full` |
| κ(T) from phonon BTE | `kinetic-evolution-of` | `BTE-RTA` (phonons) |
| κ(T) from Green-Kubo | `linear-response` | `Kubo` |
| Hot-carrier T_e steady state | `algebraic-combination` (steady-state closed form) or `kinetic-evolution-of` (transient) | `BTE-RTA` energy moment |
| Avalanche M integral | `spectral-aggregate-of` over field-on-grid | new sub-method `field-line-integral` |
| Schottky J-V | `response-of-to` | new sub-method `interface-tunneling` |
| Poiseuille / second sound | `kinetic-evolution-of` | `BTE-full` with split kernel |

**Two genuinely new sub-methods** (within existing primitives, not new primitives): `field-line-integral` and `interface-tunneling`. Both are bounded — they don't bring new mathematical capability, only specialize an existing primitive to a recurring shape.

### J.4 Residual categories

The existing 5 categories (EOM-violation, degeneracy, conservation, positivity, algebraic-identities) need:

**Positivity** — add four entries:
- `kappa-non-negative`
- `mobility-non-negative`
- `T-e-geq-T-L` (hot-electron temperature cannot dip below lattice)
- `multiplication-finite` (M < ∞ ⇔ ∫α dx < 1; squared barrier residual)

**Conservation** — add:
- `energy-balance-joule-heat` (∫j·E dV = ∫C_p ρ ∂T/∂t dV + ∫(κ∇T)·n̂ dA, the integral form of the heat PDE)
- `current-continuity` (∇·j + ∂ρ/∂t = 0; already implicit but worth pinning)

**Algebraic-identities** — add:
- `method-equivalence-chynoweth-bte` (cheap-α vs faithful-α match in high-field limit)
- `method-equivalence-caughey-thomas-bte` (cheap-μ(E) vs faithful-μ(E) match)
- `method-equivalence-slack-bte-kappa` (cheap-κ(T) vs faithful-κ(T) match)
- `einstein-relation-at-T` (already present in base) extended to validate at T_e ≠ T_L
- `wiedemann-franz-at-T` — already present (`lorenz-wiedemann-franz` formula); use as identity residual against faithful Kubo

**EOM-violation** — recommendation: **add a new sub-residual `coupled-em-thermal-pde-residual`** inside the existing `eom-violation/` directory (preferred over creating a new top-level category). Signature given in §D.4. This preserves "one primary residual category" while accommodating the emergent-field-tuple PDE check that chip-design observables most need.

If cross-stream synthesis prefers a clean new category, the alternative is `residuals/coupled-pde/` as a 6th top-level category — also valid.

### J.5 State-vector status of new quantities

Critical discipline question: do hot-electron T_e, local E-field, local current j become state aspects, or remain emergent?

| Quantity | Status | Justification |
|---|---|---|
| T_e | **Emergent** | Second moment of f_n(k) — derivable from existing γ̂ / emergent carrier distribution. Adding to state would double-count. |
| E_field(r) | **Already in state** | Derivable from `A(r,t)` already in unified state via E = −∂A/∂t − ∇φ; φ is recovered from γ̂ via Poisson. No addition needed. |
| j(r) | **Emergent** | First moment of f_n(k). *(Correction 2026-07-21: the claim that a current-density-from-distribution row is "already in registry" was false — no such row exists. The moment itself is emergent and needs no state addition; the conclusion stands, the justification did not.)* |
| T_L (lattice T) | **Already in state** | Encoded in vibrational entropy term S_vib[x] of S functional. |
| Local strain ε(r) | **Already in state** | Function of (h, R_I) deviation from reference. |
| Defect populations from cyclic loading | **Discrete state aspect** | If treated, lives in `SiteDecoration` `tag = 'defect` with `occupancy`. |

**No additions to the unified state vector required.** All Part-A–I physics is emergent from the existing seven-tuple `(h, R_I, P_I, Π_h, Z_I, γ̂, A)`. This is a clean result: the GENERIC framework as currently scoped is already expressive enough for the UWBG-chip-at-T problem.

### J.6 Two-tier accuracy summary

| Physics | Cheap-compute tier | Faithful-residual tier |
|---|---|---|
| μ(E) | Caughey-Thomas closed form | BTE-RTA j(E) |
| v_sat | Closed-form POP limit | BTE high-field asymptote |
| α(E) | Chynoweth a·exp(−b/E) | BTE distribution tail |
| κ(T) | Slack fit | Green-Kubo / phonon BTE-RTA |
| T_e | Steady-state closed form | BTE second moment, transient |
| J_FN | FN closed form | NEGF tunneling probability |
| TBR | AMM / DMM | Atomistic Green's function (out of scope) |
| Coupled PDE | DD + heat PDE finite-volume | BTE + full Poisson + heat self-consistent |

Each row has the **algebraic-identities method-equivalence residual** that ties cheap to faithful — this is the contract that makes the two-tier scheme verifiable.

### J.7 Material coverage check

Diamond, c-BN, h-BN, AlN, GaN, β-Ga₂O₃, AlGaN ternary, refractory metals, dielectrics: all anchored by at least one numeric value per formula above. The only sparse spot is h-BN, where anisotropy means transport coefficients are tensor-anisotropic from the start (in-plane vs out-of-plane); recommend the `tensor-indexed` bundle handles this naturally, no additional formula needed.
