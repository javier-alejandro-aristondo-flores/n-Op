### A.3 Equilibrium concentration

**Cheap path**: dilute Boltzmann
```
[D]^q_cheap(T) = N_site В· g_q В· exp(-E_form^q / kT)
```

**Faithful path**: include vibrational entropy + temperature-dependent E_F from charge balance
```
[D]^q_faith(T) = N_site В· g_q В· exp(S_form^q / k) В· exp(-(E_form^q(E_F(T), Ој(T))) / kT)
```
with E_F(T) solved self-consistently from
вҲ‘_D вҲ‘_q qВ·[D]^q + p(E_F, T) вҲ’ n(E_F, T) + вҲ‘_dopant q_d В· [Dopant^q_d] = 0.

### A.4 Worked example: V_C in diamond at 500В°C (T = 773 K)

E_form(V_C^0, intrinsic) вүҲ 7.2 eV (HSE06). kT = 0.0666 eV. N_site = 1.76e23 cmвҒ»Ві.

[V_C] вүҲ 1.76e23 В· exp(вҲ’108) вүҲ vanishing (~1eвҲ’24 cmвҒ»Ві). **At thermal equilibrium V_C is negligible**; harsh-environment generation comes from non-equilibrium sources: ion-cascade displacement (turbine ingestion), interface-driven generation (carbide-front advance, Part G), and frozen-in concentrations from synthesis. The PINO should treat [V_C] as a state variable Z_I evolving by generationвҖ“annihilation kinetics, not an equilibrium quantity.

### B.1 Per-host dopant table

|---|---|---|---|---|

### B.2 Why diamond doping is hard

Five compounding obstacles, each worth a typed term:

1. **Small lattice constant** (a = 3.567 Г…) вҶ’ tiny substitutional cavity вҶ’ most dopants prefer interstitial sites where they're deep donors or kinetically frozen.
2. **High cohesive energy** (~7.4 eV/atom) вҶ’ small formation-energy window before competing phases (graphite, dopant precipitates) win.
3. **No native shallow donor**: P at 0.57 eV gives <0.1% activation at RT; usable only at high T (favorable for harsh-env!).
4. **Self-compensation**: substitutional B^вҲ’ pairs with V_C^+ to form BвҖ“V; N^+_s pairs with V_C^вҲ’ to form NV^0.
5. **Surface depletion**: H-termination's NEA strips electrons; O-termination depletes holes. Bulk doping efficacy depends on surface boundary condition.

### B.3 Typed signatures

```

DX_transition     : (Host, Dopant, x_alloy) вҶ’ E_DX_onset
```

**Cheap path** (single-donor model):
```

```
**Faithful path**: multi-level charge-balance with all defects from Part A, T-dependent Ој (Part C), and finite Оә via Freysoldt.

### B.4 Worked example: P-doped diamond at 500В°C

E_d = 0.57 eV, T = 773 K, kT = 0.0666 eV, N_D = 1e19 cmвҒ»Ві, N_C(773 K) вүҲ 4e19 cmвҒ»Ві.
exp(вҲ’E_d/kT) вүҲ exp(вҲ’8.56) вүҲ 1.9eвҲ’4 вҮ’ nвӮҒ вүҲ 7.6e15 cmвҒ»Ві. Activation fraction (freeze-out limit) вүҲ вҲҡ(N_C/(g_dВ·N_D)) В· exp(вҲ’E_d/2kT) вүҲ 1.41 В· 0.014 вүҲ ~2%.
n вүҲ 2e17 cmвҒ»Ві at 500В°C вҖ” usable. Same dopant at RT gives ~1e14 cmвҒ»Ві вҖ” useless. **Harsh-environment T turns marginal dopants into viable ones**; this is a real PINO-relevant inversion.

## Part C вҖ” Charged-defect machinery

### C.1 ZhangвҖ“Northrup formation energy

```
E_form[D^q] = E_tot[D^q] вҲ’ E_tot[host] вҲ’ вҲ‘_i n_i В· Ој_i + qВ·(E_VBM + E_F) + E_corr[q, L]
```
Types:
```
E_form : (DefectConfig, charge q, Ој_env : List[(species, Ој)],
          E_F : Real, supercell_size L : Length, corr_scheme : Tag) вҶ’ Energy
```
- Ој_i bounded by stability triangle (e.g., Ој_C вҲҲ [Ој_C(diamond) вҲ’ О”H_f(competing phase), Ој_C(diamond)])
- E_F sweeps [0, E_g]
- E_corr handles spurious supercell electrostatics.

### C.2 MakovвҖ“Payne (cheap path)

Leading multipole correction for charged supercell with compensating jellium:
```
E_MP(q, L) = (qВІ В· Оұ_M) / (2 В· Оө В· L) + (2ПҖ В· q В· Q_r) / (3 В· Оө В· LВі) + O(LвҒ»вҒө)
sign convention: ADDED to E_tot[D^q] to recover dilute limit
```
Type:
```
E_MP : (charge q, supercell_L, Оұ_Madelung, Оө_static, Q_r : QuadrupoleMoment?) вҶ’ Energy
```
**Status in /physics**: cheap-tier correction; Оұ_M precomputed per Bravais lattice (diamond fcc вҶ’ Оұ_M вүҲ 1.418).

### C.3 Freysoldt (faithful path)

Splits the correction into **point-charge lattice term** + **potential alignment**:
```
E_corr^FNV = E_lat(q, Оө, L) вҲ’ qВ·О”V_q/0     -- О”V_q/0 = вҹЁV_defectвҹ© вҲ’ вҹЁV_pristineвҹ© far from defect
                                           -- (symbol E_corr, not E_F: E_F is the Fermi level everywhere else)
E_lat(q, Оө, L) = (qВІВ·Оұ_M)/(2В·ОөВ·L) В· [1 вҲ’ f_shape(L, Пғ_model)]
```
Type:
```
E_Frey : (q, L, Оө_вҲһ, Оө_0, model_charge_profile, V_diff_planar_avg : Array1D) вҶ’ Energy
О”V_alignment : (V_defect : ScalarField3D, V_host : ScalarField3D, r_sample : Region) вҶ’ Energy
```
Returns both the energy correction AND the alignment scalar (an alignment-residual observable for the PINO).

### C.4 LanyвҖ“Zunger

Empirical scaling of M-P for screening + image-charge interactions:
```
E_LZ(q, L) = (1 вҲ’ c_sh) В· (qВІВ·Оұ_M)/(2В·ОөВ·L),   c_sh вүҲ 0.36 for cubic
```
Cheaper than Freysoldt, better than bare M-P for cubic hosts.

### C.5 DefectвҖ“defect interaction & configurational entropy

At [D] вүі 1e19 cmвҒ»Ві, dilute-Boltzmann breaks. Add pairwise screened-Coulomb term:
```
E_dd(q_i, q_j, r_ij, Оө) = (q_iВ·q_jВ·eВІ) / (4ПҖВ·ОөВ·r_ij) В· exp(вҲ’r_ij/О»_D)
О»_D = вҲҡ(ОөВ·kT / (eВІВ·вҲ‘_q qВІВ·[D^q]))    -- Debye screening length, self-consistent
```
Configurational entropy on N_site sublattice:
```
S_conf = kВ·[N_siteВ·ln(N_site) вҲ’ [D]В·ln([D]) вҲ’ (N_site вҲ’ [D])В·ln(N_site вҲ’ [D])]
       вүҲ вҲ’kВ·[D]В·(ln([D]/N_site) вҲ’ 1)   in dilute limit
```

### C.6 Two-tier ladder for charged defects

| Tier | Formation energy | Correction | Cost |
|---|---|---|---|
| cheap | DFT-GGA E_tot, M-P only | E_MP | 1Г— |
| mid | DFT-HSE E_tot, LanyвҖ“Zunger | E_LZ | ~30Г— |
| faithful | DFT-HSE + GW band edges, Freysoldt with planar-avg alignment | E_Frey + О”V | ~300Г— |

PINO surrogate trained on cheap, residual-corrected against mid, anchor-validated against faithful.

## Part D вҖ” Recombination and trapping

### D.1 ShockleyвҖ“ReadвҖ“Hall

```
R_SRH = (nВ·p вҲ’ n_iВІ) / (П„_pВ·(n + n_1) + П„_nВ·(p + p_1))
n_1 = N_C В· exp(вҲ’(E_C вҲ’ E_T)/kT),   p_1 = N_V В· exp(вҲ’(E_T вҲ’ E_V)/kT)
П„_n = 1/(Пғ_n В· v_th В· N_T),   П„_p = 1/(Пғ_p В· v_th В· N_T)
v_th(T) = вҲҡ(3kT/m*)
```
Type:
```
R_SRH : (n, p, T, traps : List[Trap]) вҶ’ Rate
Trap = { level E_T, Пғ_n(T), Пғ_p(T), concentration N_T, degeneracy g_T }
```

### D.2 Auger

```
R_Aug = (C_n В· n + C_p В· p) В· (nВ·p вҲ’ n_iВІ)
C_n, C_p : T-dependent, ~1eвҲ’30 cmвҒ¶/s for wide-gap (much smaller than narrow-gap)
```
Relevant in diamond power devices at high injection.

### D.3 Multiphonon emission (deep-level non-radiative)

HuangвҖ“Rhys formalism for capture at deep trap:
```
Пғ_capture(T) = Пғ_0 В· exp(вҲ’E_b / kT)
E_b = (S вҲ’ О”E/в„ҸПү)ВІ В· в„ҸПү / (4S)     -- effective barrier from Marcus-style crossing
S = HuangвҖ“Rhys factor (dimensionless lattice coupling)
в„ҸПү = effective phonon energy
```
Type:
```
Пғ_MPE : (Trap, T, S : HuangRhys, Пү_phonon, О”E : energy_release) вҶ’ CrossSection
```
**Cheap path**: temperature-independent Пғ_0 from device data.
**Faithful path**: compute S and Пү from DFT phonon mode at defect; Marcus crossing for barrier.

### D.4 Capture cross sections

Three regimes:
- Coulomb-attractive (trap charge opposite carrier): Пғ ~ 1eвҲ’14 cmВІ, Пғ вҲқ T^вҲ’2
- Neutral: Пғ ~ 1eвҲ’15 cmВІ, weak T-dependence
- Coulomb-repulsive: Пғ ~ 1eвҲ’18 cmВІ В· exp(вҲ’E_C/kT), strongly T-activated

```
Пғ_capture_type : (Trap_charge_relative_to_carrier) вҶ’ {Attractive | Neutral | Repulsive}
Пғ : (type, T, Z_eff) вҶ’ CrossSection
```

### D.5 Diamond-specific recombination centers at 500В°C

NV^0/NV^вҲ’ has thermal activation ~1.7 eV вҶ’ becomes a dominant SRH center at T > 600 K. The N concentration (in CVD diamond, 1e17вҖ“1e19 cmвҒ»Ві) sets П„_n via NV trap density. Boron-doped diamond at 500В°C with [N]=1e18 cmвҒ»Ві gives П„_n вүҲ 10вҖ“100 ns (rough estimate, Пғ_n ~ 1eвҲ’15 cmВІ, v_th вүҲ 4e7 cm/s at 773 K).

### E.3 Dangling-bond states

Bare diamond surfaces have unsaturated spВі вҶ’ in-gap states near E_F. Reconstructions (ПҖ-bonded chains on (111), dimers on (100)) push these toward band edges but leave residual states near mid-gap. These act as Fermi-level pinning sites for unpassivated contacts.

```
ПҒ_DOS_surface(E) : (Facet, Termination) вҶ’ DOS_in_gap
```

### E.4 NEA/PEA mechanism for diamond

```
ПҮ_surface = ПҮ_bulk + О”V_surf_dipole
О”V_surf_dipole = (Ој_dipole В· n_dipole) / (Оө_0 В· Оө_r_surface)
```
H-termination: C^ОҙвҲ’вҖ“H^Оҙ+ dipole pointing out вҶ’ reduces ПҮ by ~1.8 eV вҶ’ NEA.
O-termination: C^Оҙ+вҖ“O^ОҙвҲ’ dipole pointing in вҶ’ increases ПҮ by ~1.5 eV вҶ’ PEA.
**Reliability concern**: H desorbs at ~700вҖ“900В°C вҶ’ NEA contacts degrade at harsh-env T. O-termination is thermally more stable. This is a measurable PINO observable: ПҮ(T, t) under operating conditions.

### E.5 Companion-UWBG surface terminations (one-liners)

- **c-BN**: (110) cleavage face, B-rich vs N-rich (100), H/F passivation common
- **AlN**: (0001) Al-polar vs (000-1) N-polar, polarity governs polarization charge
- **GaN**: same Al/N polarity story; Ga-polar (0001) is the engineered surface
- **ОІ-GaвӮӮOвӮғ**: (-201), (010), (100) all distinct; (010) is the cleavage plane
- **AlGaN**: inherits GaN polarity, with composition-dependent surface segregation

## Part F вҖ” MetalвҖ“semiconductor interfaces

### F.1 SchottkyвҖ“Mott (cheap path)

```
ПҶ_B^n = ПҶ_M вҲ’ ПҮ_S        -- n-type barrier
ПҶ_B^p = E_g + ПҮ_S вҲ’ ПҶ_M   -- p-type barrier
ПҶ_B^n + ПҶ_B^p = E_g
```
Type:
```
ПҶ_B_SM : (ПҶ_M : eV, ПҮ_S : eV) вҶ’ eV
```
**Applies when**: no MIGS, no interface dipole, no Fermi-pinning. Almost never true; useful as zeroth-order estimate.

### F.2 Bardeen limit

```
ПҶ_B вҶ’ E_g вҲ’ ПҶ_CNL   -- independent of ПҶ_M
```
Holds when interface-state density D_it is so large (>1e14 cmвҒ»ВІВ·eVвҒ»В№) that the Fermi level pins regardless of metal.

### F.3 MIGS / CowleyвҖ“Sze (faithful path)

```
ПҶ_B = S В· (ПҶ_M вҲ’ ПҮ_S) + (1 вҲ’ S) В· (E_g вҲ’ ПҶ_CNL)
S = 1 / (1 + (eВІВ·D_itВ·Оҙ) / (Оө_iВ·Оө_0))
```
S = slope parameter (S=1 вҶ’ S-M; S=0 вҶ’ Bardeen).
ПҶ_CNL = charge neutrality level of MIGS continuum.
Оҙ = interface-layer thickness; Оө_i = interface permittivity.

Type:
```
ПҶ_B_MIGS : (ПҶ_M, ПҮ_S, E_g, ПҶ_CNL, D_it, Оҙ, Оө_i) вҶ’ eV
S_param  : (D_it, Оҙ, Оө_i) вҶ’ DimensionlessFraction
```

### F.6 Thermal stability ranking (вүҘ500В°C harsh-env contacts)

Best вҶ’ worst:
1. **Pt** (no carbide, high ПҶ_M, expensive)

## Part G вҖ” High-T degradation pathways

All Arrhenius-form: `rate = ОҪвӮҖ В· exp(вҲ’E_a / kT)`. Types:

### G.1 Vacancy generation

### G.2 Dopant diffusion

```

profile evolution: вҲӮC/вҲӮt = вҲҮВ·(D вҲҮC) вҲ’ вҲҮВ·(Ој_drift В· C В· E)
```
| Dopant in diamond | E_diff (eV) | D at 500В°C (cmВІ/s) | profile shift in 1000 h |
|---|---|---|---|
| B substitutional | 4.5 | ~1eвҲ’24 | <1 nm (stable) |

### G.5 Electromigration

For 500В°C operation at j=1e6 A/cmВІ, refractory metals give MTTF ~10ВівҖ“10вҒҙ h; Al fails in hours.

### H.4 Interface chemistry

- **diamond/SiC**: stable; thin amorphous-C transition layer ~1вҖ“2 nm; no carbide reaction (SiC already carbide).
- **diamond/Si**: SiC layer forms at growth T (~800В°C), 5вҖ“20 nm thick; provides nucleation but is a thermal-expansion mismatch site.
- **diamond/Ir**: no carbide; pure mechanical interface; basis of "single-crystal diamond on Ir" route.
- **diamond/Pt**: no carbide; metallic adhesion via d-band hybridization; weak but clean.

### H.5 Thermal-cycle stability

Diamond/SiC and diamond/Ir survive >1000 cycles to 500В°C; diamond/Si fails by SiC interlayer cracking within ~100 cycles at О”T=500 K.
